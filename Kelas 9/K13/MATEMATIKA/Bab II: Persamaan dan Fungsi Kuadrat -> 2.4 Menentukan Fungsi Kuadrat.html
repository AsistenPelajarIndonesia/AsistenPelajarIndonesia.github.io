Menentukan Fungsi Kuadrat dari Informasi yang Diberikan

Dalam matematika, fungsi kuadrat adalah fungsi polinomial yang memiliki derajat tertinggi 2. Bentuk umumnya adalah f(x) = ax² + bx + c, di mana a, b, dan c adalah konstanta, dengan a ≠ 0. Fungsi kuadrat ini dapat digambarkan dalam bentuk parabola.

Sebelumnya, kita telah mempelajari cara menggambar grafik fungsi kuadrat, menemukan titik puncak, titik potong dengan sumbu-x dan sumbu-y, serta sumbu simetri. Pada bagian ini, kita akan fokus pada kebalikannya: bagaimana cara menentukan fungsi kuadrat jika kita diberi informasi-informasi tertentu mengenai grafiknya.

Informasi Kunci untuk Menentukan Fungsi Kuadrat:

  • Beberapa titik koordinat yang dilalui grafik fungsi kuadrat.
  • Titik potong grafik fungsi kuadrat dengan sumbu-x (akar-akar persamaan kuadrat).
  • Titik potong grafik fungsi kuadrat dengan sumbu-y.
  • Titik puncak dan sumbu simetri dari grafik fungsi kuadrat.

Langkah awal yang paling umum adalah memisalkan fungsi kuadratnya dalam bentuk umum:

f(x) = ax² + bx + c

Kemudian, informasi yang diberikan akan digunakan untuk membentuk sistem persamaan linear untuk mencari nilai konstanta a, b, dan c.

Secara umum, untuk menentukan sebuah fungsi kuadrat secara unik, kita memerlukan setidaknya tiga informasi independen (seperti tiga titik koordinat, atau titik puncak dan satu titik lain, atau titik potong sumbu-x dan satu titik lain).

Cara Menentukan Fungsi Kuadrat Berdasarkan Informasi yang Diberikan

1. Jika Diketahui Tiga Titik Koordinat yang Berbeda

Jika kita diberi tiga titik koordinat yang dilalui oleh grafik fungsi kuadrat, misalnya (x₁, y₁), (x₂, y₂), dan (x₃, y₃), kita dapat mensubstitusikan setiap titik ke dalam bentuk umum f(x) = ax² + bx + c untuk membentuk tiga persamaan linear.

Misalnya, untuk titik (x₁, y₁), kita dapatkan:

y₁ = ax₁² + bx₁ + c

Dengan cara yang sama untuk dua titik lainnya, kita akan memperoleh sistem tiga persamaan linear dengan tiga variabel (a, b, c). Sistem persamaan ini kemudian dapat diselesaikan untuk menemukan nilai a, b, dan c.

Contoh: Melalui Tiga Titik

Tentukan fungsi kuadrat yang grafiknya melalui titik (0, 1), (1, 3), dan (2, 7).

  1. Misalkan fungsi kuadratnya adalah f(x) = ax² + bx + c.
  2. Substitusikan titik (0, 1):

    f(0) = a(0)² + b(0) + c = 1

    Diperoleh c = 1.

    Jadi, fungsinya menjadi f(x) = ax² + bx + 1.

  3. Substitusikan titik (1, 3):

    f(1) = a(1)² + b(1) + 1 = 3

    a + b + 1 = 3

    Diperoleh persamaan 1: a + b = 2

  4. Substitusikan titik (2, 7):

    f(2) = a(2)² + b(2) + 1 = 7

    4a + 2b + 1 = 7

    Diperoleh persamaan 2: 4a + 2b = 6 (atau 2a + b = 3 setelah disederhanakan).

  5. Selesaikan sistem persamaan:

    Persamaan 1: a + b = 2

    Persamaan 2: 2a + b = 3

    Kurangkan Persamaan 1 dari Persamaan 2:

    (2a + b) - (a + b) = 3 - 2

    a = 1

    Substitusikan nilai a = 1 ke Persamaan 1:

    1 + b = 2

    b = 1

  6. Jadi, nilai a = 1, b = 1, dan c = 1. Fungsi kuadratnya adalah f(x) = x² + x + 1.

2. Jika Diketahui Titik Potong Sumbu-x (Akar-akar)

Jika grafik fungsi kuadrat memotong sumbu-x di titik (p, 0) dan (q, 0), ini berarti p dan q adalah akar-akar dari persamaan kuadrat ax² + bx + c = 0. Dalam kasus ini, bentuk fungsi kuadrat dapat dituliskan dalam bentuk faktor:

f(x) = a(x - p)(x - q)

Nilai a dapat ditentukan jika kita diberi informasi tambahan, misalnya satu titik lain yang dilalui grafik, atau titik potong sumbu-y.

Contoh: Titik Potong Sumbu-x dan Satu Titik Lain

Tentukan fungsi kuadrat yang memotong sumbu-x di (-2, 0) dan (3, 0), serta melalui titik (0, 3).

  1. Karena memotong sumbu-x di (-2, 0) dan (3, 0), maka akar-akarnya adalah -2 dan 3.
  2. Gunakan bentuk faktor:

    f(x) = a(x - (-2))(x - 3)

    f(x) = a(x + 2)(x - 3)

  3. Karena melalui titik (0, 3), maka f(0) = 3. Substitusikan ke dalam fungsi:

    f(0) = a(0 + 2)(0 - 3) = 3

    a(2)(-3) = 3

    -6a = 3

    a = -1/2

  4. Substitusikan nilai a kembali ke bentuk faktor:

    f(x) = -1/2 (x + 2)(x - 3)

    Untuk mendapatkan bentuk ax² + bx + c, kita jabarkan:

    f(x) = -1/2 (x² - 3x + 2x - 6)

    f(x) = -1/2 (x² - x - 6)

    f(x) = -1/2 x² + 1/2 x + 3

3. Jika Diketahui Titik Puncak dan Sumbu Simetri

Titik puncak dari parabola adalah koordinat titik tertinggi atau terendah dari grafik. Jika titik puncaknya adalah (s, t), maka sumbu simetrinya adalah garis vertikal x = s.

Ada dua bentuk umum yang bisa digunakan ketika titik puncak diketahui:

  • Bentuk Puncak: f(x) = a(x - s)² + t. Dengan mengetahui satu titik lain yang dilalui, kita dapat mencari nilai a.
  • Jika kita mengetahui sumbu simetri x = s dan satu titik potong sumbu-x, misalnya (p, 0), maka titik potong sumbu-x yang lain dapat dicari dengan memanfaatkan sifat simetri. Titik tersebut adalah (2s - p, 0). Setelah mengetahui kedua titik potong sumbu-x dan satu titik lain (misalnya titik puncak itu sendiri atau titik potong sumbu-y), kita bisa menggunakan metode sebelumnya.

Contoh: Titik Puncak dan Sumbu Simetri

Tentukan fungsi kuadrat yang memiliki titik puncak (-1, 3) dan memotong sumbu-y di (0, 1).

  1. Titik puncak adalah (-1, 3), sehingga s = -1 dan t = 3.
  2. Gunakan bentuk puncak:

    f(x) = a(x - s)² + t

    f(x) = a(x - (-1))² + 3

    f(x) = a(x + 1)² + 3

  3. Karena memotong sumbu-y di (0, 1), maka f(0) = 1. Substitusikan:

    f(0) = a(0 + 1)² + 3 = 1

    a(1)² + 3 = 1

    a + 3 = 1

    a = -2

  4. Substitusikan nilai a kembali ke bentuk puncak:

    f(x) = -2(x + 1)² + 3

    Untuk mendapatkan bentuk ax² + bx + c, kita jabarkan:

    f(x) = -2(x² + 2x + 1) + 3

    f(x) = -2x² - 4x - 2 + 3

    f(x) = -2x² - 4x + 1

4. Jika Diketahui Titik Potong Sumbu-y dan Titik Puncak

Titik potong sumbu-y selalu berada di koordinat (0, c). Jika kita mengetahui titik potong sumbu-y (0, r), maka nilai c = r.

Jika kita mengetahui titik potong sumbu-y dan titik puncak (s, t), kita dapat menggunakan dua informasi ini. Titik potong sumbu-y (0, r) memberikan kita nilai c. Titik puncak (s, t) memberikan kita informasi tentang sumbu simetri x = s. Jika kita menggunakan bentuk f(x) = ax² + bx + c, maka kita sudah tahu c. Sumbu simetri x = -b/(2a) = s memberikan satu persamaan. Titik puncak (s, t) memberikan persamaan as² + bs + c = t. Dengan dua persamaan ini, kita dapat mencari a dan b.

Contoh: Titik Potong Sumbu-y dan Titik Puncak

Tentukan fungsi kuadrat yang memotong sumbu-y di (0, 3) dan memiliki titik puncak (-2, 1).

  1. Titik potong sumbu-y adalah (0, 3), sehingga c = 3.
  2. Titik puncak adalah (-2, 1), sehingga sumbu simetri adalah x = -2. Ingat bahwa sumbu simetri juga diberikan oleh x = -b/(2a). Jadi, -b/(2a) = -2, yang berarti b = 4a.
  3. Gunakan informasi titik puncak:

    f(s) = t

    f(-2) = 1

    Substitusikan x = -2, c = 3 ke dalam f(x) = ax² + bx + c:

    a(-2)² + b(-2) + 3 = 1

    4a - 2b + 3 = 1

    4a - 2b = -2

    2a - b = -1

  4. Kita punya sistem persamaan:

    1. b = 4a

    2. 2a - b = -1

    Substitusikan (1) ke (2):

    2a - (4a) = -1

    -2a = -1

    a = 1/2

  5. Cari nilai b menggunakan b = 4a:

    b = 4(1/2) = 2

  6. Jadi, nilai a = 1/2, b = 2, dan c = 3. Fungsi kuadratnya adalah f(x) = 1/2 x² + 2x + 3.

5. Penentuan Titik Potong Antar Dua Grafik

Menentukan titik potong dua grafik, baik linear maupun kuadrat, dilakukan dengan menyamakan kedua fungsi tersebut.

Contoh: Titik Potong Fungsi Kuadrat dengan Fungsi Linear

Tentukan titik potong antara grafik fungsi linear y = x - 1 dengan fungsi kuadrat y = x² - 5x + 4.

  1. Samakan kedua persamaan:

    x² - 5x + 4 = x - 1

  2. Pindahkan semua suku ke satu sisi untuk membentuk persamaan kuadrat:

    x² - 5x - x + 4 + 1 = 0

    x² - 6x + 5 = 0

  3. Faktorkan persamaan kuadrat tersebut:

    (x - 1)(x - 5) = 0

  4. Dapatkan nilai x:

    x = 1 atau x = 5.

  5. Substitusikan nilai x ke salah satu persamaan (misalnya fungsi linear y = x - 1) untuk mencari nilai y:
    • Jika x = 1, maka y = 1 - 1 = 0. Titik potongnya adalah (1, 0).
    • Jika x = 5, maka y = 5 - 1 = 4. Titik potongnya adalah (5, 4).
  6. Jadi, titik potongnya adalah (1, 0) dan (5, 4).

Contoh: Titik Potong Dua Fungsi Kuadrat

Tentukan titik potong antara grafik fungsi kuadrat f₁(x) = x² - 5x + 4 dan f₂(x) = x² - 4x + 2.

  1. Samakan kedua fungsi:

    x² - 5x + 4 = x² - 4x + 2

  2. Pindahkan semua suku ke satu sisi:

    x² - 5x + 4 - x² + 4x - 2 = 0

    -x + 2 = 0

  3. Dapatkan nilai x:

    x = 2.

  4. Substitusikan nilai x ke salah satu fungsi (misalnya f₁(x)):

    f₁(2) = (2)² - 5(2) + 4 = 4 - 10 + 4 = -2

  5. Jadi, titik potongnya adalah (2, -2).
Tahukah Kamu? Dua grafik fungsi kuadrat dapat berpotongan paling banyak di dua titik. Jika dua grafik fungsi kuadrat berpotongan di lebih dari dua titik, maka kedua fungsi tersebut identik (merupakan fungsi yang sama).
