Bab II: Persamaan dan Fungsi Kuadrat

2.3 Sumbu Simetri dan Nilai Optimum

Dalam memahami fungsi kuadrat, dua konsep kunci yang sangat penting adalah sumbu simetri dan nilai optimum. Keduanya membantu kita memahami bentuk dan posisi grafik parabola, serta titik tertingginya (maksimum) atau terendahnya (minimum).

Memahami Sumbu Simetri

Sumbu simetri adalah sebuah garis vertikal yang membagi grafik fungsi kuadrat (parabola) menjadi dua bagian yang simetris atau saling bercermin. Artinya, jika kamu melipat grafik di sepanjang garis ini, kedua sisi akan saling bertemu dengan sempurna.

Untuk fungsi kuadrat dalam bentuk umum f(x) = ax² + bx + c, sumbu simetri dapat ditentukan dengan rumus:

x = -b / 2a

Di sini, 'a' dan 'b' adalah koefisien dari suku x² dan x pada fungsi kuadrat tersebut.

Jika fungsi kuadrat ditulis dalam bentuk f(x) = a(x - s)² + t (bentuk verteks), maka sumbu simetrinya adalah garis vertikal yang melewati titik (s, t), yaitu:

x = s

Memahami Nilai Optimum

Nilai optimum dari sebuah fungsi kuadrat merujuk pada nilai maksimum atau minimum dari fungsi tersebut. Nilai ini dicapai pada titik puncak parabola.

  • Jika grafik parabola terbuka ke atas (ketika koefisien a > 0), maka titik puncak adalah titik terendah. Nilai optimumnya adalah nilai minimum.
  • Jika grafik parabola terbuka ke bawah (ketika koefisien a < 0), maka titik puncak adalah titik tertinggi. Nilai optimumnya adalah nilai maksimum.

Nilai optimum ini adalah nilai y dari titik puncak. Jika sumbu simetri adalah x = -b / 2a, maka nilai optimumnya adalah f(-b / 2a).

Dalam bentuk verteks f(x) = a(x - s)² + t, nilai optimumnya sangat jelas terlihat:

Jika a > 0, nilai optimumnya adalah minimum sebesar t.

Jika a < 0, nilai optimumnya adalah maksimum sebesar t.

Jadi, nilai optimumnya adalah t, yang merupakan nilai y dari titik puncak (s, t).

Titik Puncak (Titik Optimum)

Titik puncak adalah koordinat (x, y) di mana nilai optimum terjadi. Berdasarkan sumbu simetri dan nilai optimum:

Titik Puncak = (sumbu simetri, nilai optimum)

Atau dalam bentuk umum: (-b / 2a, f(-b / 2a))

Atau dalam bentuk verteks: (s, t)

Menentukan Nilai Optimum Menggunakan Diskriminan

Kita juga bisa menentukan nilai optimum (y-koordinat titik puncak) menggunakan diskriminan (D) dari persamaan kuadrat ax² + bx + c = 0, yaitu D = b² - 4ac.

Nilai optimumnya (y-koordinat titik puncak) adalah:

y₀ = -D / 4a

Sehingga, titik optimumnya adalah (-b / 2a, -D / 4a).

Contoh Ilustrasi

Misalkan kita punya fungsi kuadrat f(x) = x² - 6x + 10.

Di sini, a = 1, b = -6, dan c = 10.

1. Menentukan Sumbu Simetri:

x = -b / 2a = -(-6) / (2 * 1) = 6 / 2 = 3. Jadi, sumbu simetrinya adalah x = 3.

2. Menentukan Nilai Optimum:

Karena a = 1 (positif), maka parabola terbuka ke atas dan memiliki nilai minimum.

Nilai minimumnya adalah f(3):

f(3) = (3)² - 6(3) + 10 = 9 - 18 + 10 = 1.

Atau menggunakan rumus y₀ = -D / 4a:

D = b² - 4ac = (-6)² - 4(1)(10) = 36 - 40 = -4.

y₀ = -(-4) / (4 * 1) = 4 / 4 = 1.

Jadi, nilai minimumnya adalah 1.

3. Menentukan Titik Optimum (Puncak):

Titik optimumnya adalah (sumbu simetri, nilai optimum) = (3, 1).

Pergeseran Grafik dan Dampaknya

Memahami pergeseran grafik dari fungsi dasar f(x) = x² sangat membantu dalam menentukan sumbu simetri dan nilai optimum.

  • Pergeseran horizontal: Fungsi f(x) = a(x - s)² akan digeser sejauh s satuan ke kanan jika s positif, dan sejauh s satuan ke kiri jika s negatif. Sumbu simetrinya menjadi x = s.
  • Pergeseran vertikal: Fungsi f(x) = a x² + t akan digeser sejauh t satuan ke atas jika t positif, dan sejauh t satuan ke bawah jika t negatif. Nilai optimumnya menjadi t.
  • Pergeseran gabungan: Fungsi f(x) = a(x - s)² + t merupakan hasil pergeseran dari f(x) = ax². Sumbu simetrinya adalah x = s dan nilai optimumnya adalah t.

Dengan menguasai konsep sumbu simetri dan nilai optimum, kita dapat menganalisis karakteristik grafik fungsi kuadrat secara mendalam, mulai dari titik puncaknya hingga arah terbukanya parabola, yang merupakan dasar penting untuk menggambar sketsa grafik fungsi kuadrat.
