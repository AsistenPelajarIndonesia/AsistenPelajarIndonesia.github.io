Bab II: Persamaan dan Fungsi Kuadrat

2.2 Grafik Fungsi Kuadrat

Memahami Bentuk Umum Fungsi Kuadrat

Fungsi kuadrat adalah fungsi yang memiliki bentuk umum:

y = ax² + bx + c

Di mana:

  • a, b, dan c adalah konstanta (bilangan real).
  • a ≠ 0. Syarat ini sangat penting! Jika a = 0, maka suku ax² akan hilang, dan fungsi tersebut berubah menjadi fungsi linear (y = bx + c), bukan lagi fungsi kuadrat.
  • x dan y adalah variabel.

Grafik dari fungsi kuadrat akan selalu berbentuk parabola. Bentuk parabola ini sangat dipengaruhi oleh nilai-nilai a, b, dan c.

Pengaruh Koefisien a terhadap Grafik

Koefisien a menentukan arah terbuka dan "kelebaran" parabola.

  • Jika a > 0 (a positif):

    Parabola akan terbuka ke atas. Semakin besar nilai a, semakin sempit (kurus) parabola tersebut. Bayangkan seperti Anda menarik ujung-ujung parabola ke atas, ia akan semakin meruncing.

  • Jika a < 0 (a negatif):

    Parabola akan terbuka ke bawah. Semakin kecil (semakin negatif) nilai a, semakin sempit (kurus) parabola tersebut. Bayangkan seperti Anda menarik ujung-ujung parabola ke bawah, ia juga akan semakin meruncing.

Contoh Perbandingan:

y = x² (parabola terbuka ke atas, relatif lebar)
y = 2x² (parabola terbuka ke atas, lebih sempit dari y = x²)
y = -x² (parabola terbuka ke bawah, relatif lebar)
y = -3x² (parabola terbuka ke bawah, lebih sempit dari y = -x²)

Titik terendah pada parabola yang terbuka ke atas disebut titik balik minimum, sedangkan titik tertinggi pada parabola yang terbuka ke bawah disebut titik balik maksimum.

Pengaruh Koefisien c terhadap Grafik

Koefisien c menentukan titik perpotongan grafik dengan sumbu-y.

  • Grafik fungsi kuadrat y = ax² + bx + c akan selalu memotong sumbu-y pada koordinat (0, c).

Ini karena ketika x = 0, maka suku ax² dan bx akan bernilai nol, sehingga y hanya akan bernilai c.

Secara visual, nilai c menggeser grafik parabola ke atas atau ke bawah tanpa mengubah bentuk dan arah terbukanya.

  • Jika c > 0, parabola bergeser ke atas sejauh c satuan dari titik (0,0).
  • Jika c < 0, parabola bergeser ke bawah sejauh |c| satuan dari titik (0,0).
Tips: Untuk menggambar grafik fungsi kuadrat, Anda bisa memulainya dengan mencari beberapa titik koordinat (x, y) dengan mensubstitusikan nilai-nilai x yang berbeda ke dalam fungsi. Titik-titik ini kemudian diplot pada bidang Kartesius.

Pengaruh Koefisien b terhadap Grafik (Titik Puncak dan Sumbu Simetri)

Koefisien b sangat memengaruhi posisi titik puncak dan sumbu simetri parabola.

  • Sumbu Simetri: Garis vertikal yang membagi parabola menjadi dua bagian yang simetris. Persamaan sumbu simetri untuk fungsi y = ax² + bx + c adalah x = -b / (2a).
  • Titik Puncak: Titik di mana parabola "berbelok". Koordinat titik puncak adalah (-b / (2a), f(-b / (2a))).

Bagaimana b mempengaruhi posisi?

  • Ketika a dan c tetap, perubahan nilai b akan menggeser sumbu simetri (dan juga titik puncak) secara horizontal.
  • Jika a > 0, titik puncak adalah titik balik minimum.
  • Jika a < 0, titik puncak adalah titik balik maksimum.

Contoh Kasus Sederhana (b ≠ 0, c = 0):

  • y = x² (a=1, b=0, c=0): Sumbu simetri x=0, titik puncak (0,0).
  • y = x² + 2x (a=1, b=2, c=0): Sumbu simetri x = -2/(2*1) = -1. Titik puncak (-1, (-1)² + 2(-1)) = (-1, 1 - 2) = (-1, -1). Parabola bergeser ke kiri.
  • y = x² - 2x (a=1, b=-2, c=0): Sumbu simetri x = -(-2)/(2*1) = 1. Titik puncak (1, (1)² - 2(1)) = (1, 1 - 2) = (1, -1). Parabola bergeser ke kanan.

Perubahan Akibat 'a'

Perhatikan bagaimana perubahan nilai 'a' mempengaruhi bentuk parabola. Grafik akan menjadi lebih sempit atau lebih lebar.

Perubahan Akibat 'c'

Perhatikan bagaimana perubahan nilai 'c' menggeser grafik ke atas atau ke bawah tanpa mengubah bentuknya.

Perubahan Akibat 'b'

Perhatikan bagaimana perubahan nilai 'b' menggeser posisi sumbu simetri dan titik puncak secara horizontal.

Kesimpulan Utama

Grafik fungsi kuadrat y = ax² + bx + c adalah parabola yang:

  1. Terbuka ke atas jika a > 0, dan ke bawah jika a < 0.
  2. Semakin sempit jika nilai absolut |a| semakin besar.
  3. Memotong sumbu-y di titik (0, c).
  4. Memiliki sumbu simetri pada x = -b / (2a).
  5. Memiliki titik puncak pada koordinat (-b / (2a), f(-b / (2a))), yang merupakan titik minimum jika a > 0 dan titik maksimum jika a < 0.
