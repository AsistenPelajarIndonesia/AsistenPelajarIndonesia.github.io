Bab II: Persamaan Kuadrat

2.1 Persamaan Kuadrat

Pengantar

Persamaan kuadrat adalah tulang punggung dalam banyak masalah matematika, baik dalam konteks teoritis maupun aplikasi praktis di kehidupan sehari-hari. Memahami bagaimana persamaan kuadrat dibentuk dan bagaimana menemukan solusinya adalah kunci untuk memecahkan berbagai tantangan.

Secara umum, persamaan kuadrat satu variabel dapat dituliskan dalam bentuk:

ax² + bx + c = 0

dengan a, b, dan c adalah konstanta real, dan yang terpenting, a ≠ 0. Jika a = 0, maka persamaan tersebut bukan lagi persamaan kuadrat, melainkan menjadi persamaan linear.

Bilangan a, b, dan c dikenal sebagai koefisien dari persamaan kuadrat.

Menemukan Akar Persamaan Kuadrat

Akar-akar atau solusi dari sebuah persamaan kuadrat adalah nilai-nilai variabel (biasanya x) yang membuat persamaan tersebut bernilai benar (sama dengan nol). Ada tiga metode utama yang sering digunakan untuk menemukan akar-akar persamaan kuadrat:

1. Memfaktorkan

Metode ini melibatkan penulisan ulang persamaan kuadrat sebagai hasil kali dua faktor linear. Jika kita memiliki persamaan dalam bentuk ax² + bx + c = 0, kita mencoba mencari dua bilangan, katakanlah p dan q, sedemikian rupa sehingga persamaan dapat difaktorkan menjadi (ax + p)(bx + q) = 0 atau bentuk yang setara.

Prinsip dasarnya adalah jika hasil kali dua bilangan adalah nol, maka salah satu atau kedua bilangan tersebut harus nol. Jadi, jika (x - r₁)(x - r₂) = 0, maka x - r₁ = 0 atau x - r₂ = 0, yang memberikan solusi x = r₁ dan x = r₂.

Contoh Sederhana: Faktorisasi

Misalkan kita punya persamaan x² + 5x + 6 = 0.

Kita perlu mencari dua bilangan yang jika dikalikan menghasilkan 6 dan jika dijumlahkan menghasilkan 5. Bilangan tersebut adalah 2 dan 3.

Maka, persamaan dapat difaktorkan menjadi:

(x + 2)(x + 3) = 0

Ini berarti x + 2 = 0 atau x + 3 = 0.

Sehingga, akar-akarnya adalah x = -2 dan x = -3.

2. Melengkapi Kuadrat Sempurna

Metode ini mengubah persamaan kuadrat yang diberikan menjadi bentuk kuadrat sempurna, yaitu (x + p)² = q. Ini sangat berguna ketika pemfaktoran tidak mudah dilakukan.

Langkah-langkah umumnya adalah:

  1. Pastikan koefisien x² adalah 1. Jika tidak, bagi seluruh persamaan dengan koefisien x² tersebut.
  2. Pindahkan konstanta (suku c) ke sisi kanan persamaan.
  3. Tambahkan kuadrat dari setengah koefisien x (yaitu, (b/2)²) ke kedua sisi persamaan. Ini akan membuat sisi kiri menjadi kuadrat sempurna.
  4. Ambil akar kuadrat dari kedua sisi. Ingat untuk menyertakan tanda positif dan negatif.
  5. Selesaikan untuk x.
Contoh Sederhana: Melengkapi Kuadrat Sempurna

Misalkan kita punya persamaan x² + 6x + 5 = 0.

1. Koefisien x² sudah 1.

2. Pindahkan konstanta: x² + 6x = -5.

3. Setengah koefisien x adalah 6/2 = 3. Kuadratnya adalah 3² = 9. Tambahkan 9 ke kedua sisi:

x² + 6x + 9 = -5 + 9
(x + 3)² = 4

4. Ambil akar kuadrat: x + 3 = ±√4, sehingga x + 3 = ±2.

5. Selesaikan untuk x:

x + 3 = 2 → x = -1

x + 3 = -2 → x = -5

3. Rumus Kuadratik (Rumus ABC)

Rumus kuadratik adalah metode yang paling umum dan dapat digunakan untuk menyelesaikan semua jenis persamaan kuadrat. Rumus ini diturunkan dari metode melengkapi kuadrat sempurna dengan menerapkan pada bentuk umum ax² + bx + c = 0.

Rumusnya adalah:

x₁,₂ = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}

Di sini, ± menunjukkan bahwa ada dua kemungkinan solusi: satu menggunakan tanda tambah dan satu lagi menggunakan tanda kurang.

Contoh Sederhana: Rumus Kuadratik

Misalkan kita punya persamaan 2x² + 7x + 3 = 0.

Di sini, a = 2, b = 7, dan c = 3.

Masukkan nilai-nilai ini ke dalam rumus:

x₁,₂ = \frac{-7 \pm \sqrt{7^2 - 4(2)(3)}}{2(2)}
x₁,₂ = \frac{-7 \pm \sqrt{49 - 24}}{4}
x₁,₂ = \frac{-7 \pm \sqrt{25}}{4}
x₁,₂ = \frac{-7 \pm 5}{4}

Sekarang, kita pisahkan untuk kedua solusi:

Solusi 1 (menggunakan +): x₁ = \frac{-7 + 5}{4} = \frac{-2}{4} = -\frac{1}{2}

Solusi 2 (menggunakan -): x₂ = \frac{-7 - 5}{4} = \frac{-12}{4} = -3

Jadi, akar-akarnya adalah x = -1/2 dan x = -3.

Diskriminan dan Sifat Akar

Dalam rumus kuadratik, bagian di bawah akar kuadrat, yaitu b² - 4ac, memiliki peran krusial. Bagian ini disebut Diskriminan dan dilambangkan dengan D.

D = b² - 4ac

Nilai diskriminan (D) menentukan sifat dari akar-akar persamaan kuadrat:

  • Jika D > 0, maka persamaan kuadrat memiliki dua akar real yang berbeda.
  • Jika D = 0, maka persamaan kuadrat memiliki satu akar real (akar kembar).
  • Jika D < 0, maka persamaan kuadrat tidak memiliki akar real (akarnya adalah bilangan imajiner/kompleks).
Analisis Diskriminan

Mari kita lihat beberapa contoh:

  1. Persamaan: x² + 5x + 6 = 0

    2. a=1, b=5, c=6

    D = 5² - 4(1)(6) = 25 - 24 = 1

    Karena D = 1 > 0, maka memiliki dua akar real yang berbeda (-2 dan -3).

  2. Persamaan: x² + 6x + 9 = 0

    3. a=1, b=6, c=9

    D = 6² - 4(1)(9) = 36 - 36 = 0

    Karena D = 0, maka memiliki satu akar real kembar (-3).

  3. Persamaan: x² + 2x + 5 = 0

    4. a=1, b=2, c=5

    D = 2² - 4(1)(5) = 4 - 20 = -16

    Karena D = -16 < 0, maka tidak memiliki akar real.

Hubungan Akar dan Koefisien

Untuk persamaan kuadrat dalam bentuk ax² + bx + c = 0, jika x₁ dan x₂ adalah akar-akarnya, maka berlaku hubungan menarik antara akar-akar tersebut dengan koefisien-koefisiennya:

  • Jumlah Akar-akar:
    • x₁ + x₂ = \frac{-b}{a}
  • Hasil Kali Akar-akar:
    • x₁ * x₂ = \frac{c}{a}

Hubungan ini sangat berguna untuk mengecek kebenaran solusi atau untuk membentuk persamaan kuadrat baru jika akar-akarnya diketahui.

Penerapan Hubungan Akar

Untuk persamaan 2x² + 7x + 3 = 0, kita tahu akarnya adalah x₁ = -1/2 dan x₂ = -3. Mari kita cek:

Jumlah Akar: x₁ + x₂ = -\frac{1}{2} + (-3) = -\frac{1}{2} - \frac{6}{2} = -\frac{7}{2}. Sesuai dengan \frac{-b}{a} = \frac{-7}{2}.

Hasil Kali Akar: x₁ * x₂ = (-\frac{1}{2}) * (-3) = \frac{3}{2}. Sesuai dengan \frac{c}{a} = \frac{3}{2}.

Aplikasi Persamaan Kuadrat dalam Kehidupan Nyata

Persamaan kuadrat sering muncul dalam masalah-masalah dunia nyata, seperti:

  • Menghitung lintasan proyektil (misalnya bola yang dilempar).
  • Menentukan luas atau dimensi suatu objek jika diketahui hubungan antar sisinya.
  • Optimasi dalam berbagai bidang seperti bisnis dan teknik.
  • Analisis sirkuit listrik.
Contoh Soal Cerita

Sebuah taman kota berbentuk persegi panjang memiliki keliling 90 m dan luas 450 m². Berapa panjang dan lebar taman tersebut?

Misalkan panjang = p meter, dan lebar = l meter.

Dari keliling: 2(p + l) = 90 → p + l = 45 → l = 45 - p.

Dari luas: p * l = 450.

Substitusikan l ke dalam persamaan luas:

p(45 - p) = 450

Bentuk ini menjadi persamaan kuadrat:

45p - p² = 450
p² - 45p + 450 = 0

Dengan memfaktorkan (mencari dua bilangan yang dikalikan 450 dan dijumlah -45, yaitu -15 dan -30), kita dapatkan:

(p - 15)(p - 30) = 0

Sehingga, p = 15 atau p = 30.

Jika panjang p = 15 m, maka lebar l = 45 - 15 = 30 m.

Jika panjang p = 30 m, maka lebar l = 45 - 30 = 15 m.

Jadi, dimensi taman tersebut adalah 30 meter dan 15 meter.
