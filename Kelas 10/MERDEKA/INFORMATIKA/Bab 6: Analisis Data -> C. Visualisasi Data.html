C. Visualisasi Data

Setelah kita berhasil mengumpulkan data melalui web scraping, langkah selanjutnya yang sangat krusial dalam analisis data adalah visualisasi data. Visualisasi mengubah data mentah yang seringkali berupa teks dan angka menjadi format grafis yang mudah dipahami, menarik, dan informatif. Ini membantu kita melihat pola, tren, dan hubungan yang mungkin terlewatkan jika hanya melihat data dalam bentuk tabel.

Bayangkan data lowongan pekerjaan yang kita dapatkan dari website seperti Jobs ID. Data ini mungkin tersaji dalam tabel yang rapi. Namun, untuk memahami gambaran besar, seperti lowongan mana yang paling banyak, atau rata-rata gaji untuk posisi tertentu, visualisasi adalah kunci.

Visualisasi data adalah proses menyajikan data dalam format grafis agar lebih mudah dipahami, dianalisis, dan diinterpretasikan.

Di bagian ini, kita akan menjelajahi bagaimana data yang telah kita kumpulkan melalui web scraping dapat divisualisasikan menggunakan berbagai jenis diagram. Kita akan melihat bagaimana kode Python, dengan bantuan library seperti Plotly, dapat mengubah data tekstual menjadi representasi visual yang powerful.

1. Pra-pemrosesan Data

Sebelum data dapat divisualisasikan, seringkali data tersebut perlu dibersihkan dan dipersiapkan. Proses ini disebut pra-pemrosesan data (data preprocessing). Mengapa ini penting?

  • Kualitas Data: Data mentah seringkali mengandung kesalahan, format yang tidak konsisten, atau nilai yang hilang.
  • Format yang Tepat: Agar dapat diolah oleh alat visualisasi, data harus dalam format yang sesuai (misalnya, angka harus dikenali sebagai angka, bukan teks).
  • Konsistensi: Memastikan data disajikan secara seragam.

Contoh dari teks yang perlu diproses adalah data gaji yang mungkin berisi teks seperti "Gaji Dirahasiakan" atau mengandung karakter pemisah ribuan seperti titik (misalnya, 3.000.000). Titik ini dapat diinterpretasikan sebagai desimal dalam perhitungan jika tidak diatasi.

Mari kita lihat contoh bagaimana membersihkan data gaji menggunakan Python:

```python # Asumsikan 't2' berisi teks gaji dari halaman web # Misalnya: "3.000.000" atau "Gaji Dirahasiakan" # Baris 26-27: Mendapatkan teks gaji t2 = t1[1].select("span") # Ini adalah contoh pengambilan elemen, sesuaikan dengan struktur HTML sebenarnya try: xgaji = t2[1].get_text() except: xgaji = t2[0].get_text() # Jika elemen kedua tidak ada # Baris 29: Membersihkan titik sebagai pemisah ribuan xgaji = xgaji.replace(".","") # Baris 30-32: Menangani kasus "Gaji Dirahasiakan" if (xgaji == "Gaji Dirahasiakan"): xgaji = 0 # Ganti dengan nilai numerik yang sesuai, misalnya 0 # Baris 33: Menambahkan gaji yang sudah dibersihkan ke dalam list 'gaji' gaji.append(xgaji) ```

Representasi kode program Python untuk pra-pemrosesan data gaji. Bayangkan ini adalah langkah penting sebelum membuat grafik.

Setelah data dibersihkan dan dipersiapkan (misalnya, data gaji diubah menjadi tipe numerik), barulah data siap untuk divisualisasikan.

2. Visualisasi Data: Diagram Batang (Barchart)

Diagram batang adalah salah satu cara paling umum dan efektif untuk membandingkan nilai antar kategori. Dalam kasus lowongan pekerjaan, kita bisa membandingkan gaji antar instansi atau antar posisi.

Untuk membuat visualisasi ini, kita akan menggunakan library Plotly di Python. Plotly adalah library yang sangat kuat untuk membuat visualisasi interaktif.

Pertama, kita perlu mengimpor library tersebut:

```python import plotly.express as px ```

Kemudian, kita bisa menggunakan fungsi px.bar() untuk membuat diagram batang. Mari kita lihat contoh kode yang menggabungkan data posisi dan gaji:

```python # Asumsikan 'posisi' adalah list nama posisi pekerjaan # Asumsikan 'instansi' adalah list nama instansi # Asumsikan 'gaji' adalah list nilai gaji (sudah dibersihkan dan numerik) # Membuat DataFrame menggunakan Pandas import pandas as pd lowker = pd.DataFrame({ "Posisi": posisi, "Instansi": instansi, "Gaji": gaji }) # Membuat diagram batang # Di sini, kita membandingkan Gaji berdasarkan Instansi fig = px.bar(lowker, x='Instansi', y='Gaji', title='Perbandingan Gaji per Instansi') fig.show() ```

Contoh visualisasi diagram batang yang menampilkan perbandingan gaji berdasarkan instansi. Perhatikan bagaimana setiap batang mewakili satu instansi dengan tinggi batang menunjukkan nilai gaji.

Pengelompokan Lanjutan untuk Visualisasi yang Lebih Baik

Terkadang, mengelompokkan berdasarkan satu kategori saja (misalnya, hanya instansi) bisa membuat beberapa batang bertumpuk dan sulit dibedakan. Solusinya adalah menggabungkan beberapa kategori atau memproses data lebih lanjut.

Misalnya, jika kita ingin melihat setiap lowongan pekerjaan (gabungan posisi dan instansi) secara terpisah, kita perlu memodifikasi cara data diproses sebelum visualisasi.

Berikut adalah contoh kode yang memproses dan menggabungkan informasi posisi dan instansi, lalu memvisualisasikannya:

```python # --- Bagian Pra-pemrosesan Lanjutan (menggantikan kode sebelumnya untuk posisi/instansi) --- # Asumsikan ada variasi dalam pengambilan data posisi # Contoh kode yang mungkin sedikit berbeda tergantung struktur HTML asli: posisi_dan_instansi_gabungan = [] # List untuk menyimpan gabungan posisi dan instansi # Simulasi pengambilan data (sesuaikan dengan struktur HTML sebenarnya) # Misalnya, struktur HTML memberikan judul posisi dan nama instansi terpisah # Kode di bawah ini adalah contoh konseptual untuk menggabungkan # Anda mungkin perlu menyesuaikan selector 'h3', 'p', 'a' dan logika penggabungan # Anggap kita punya list 'judul_posisi' dan 'nama_instansi' dari scraping sebelumnya # Contoh: # judul_posisi = ["French Teacher", "Social Media Specialist", ...] # nama_instansi = ["Kukche", "Ara", ...] # Iterasi untuk menggabungkan # for i in range(len(judul_posisi)): # posisi_gabung = f"{judul_posisi[i]} - {nama_instansi[i]}" # posisi_dan_instansi_gabungan.append(posisi_gabung) # --- Bagian Visualisasi --- # Membuat DataFrame baru dengan data yang sudah digabung # Anggap 'gaji' sudah dibersihkan lowker_detail = pd.DataFrame({ "PosisiDetail": posisi_dan_instansi_gabungan, # Gunakan list gabungan "Gaji": gaji }) # Membuat diagram batang dengan label yang lebih spesifik fig_detail = px.bar(lowker_detail, x='PosisiDetail', y='Gaji', title='Rincian Gaji per Lowongan (Posisi & Instansi)') fig_detail.show() ```

Contoh visualisasi diagram batang yang menampilkan rincian lowongan pekerjaan (Posisi dan Instansi) secara terpisah. Ini memungkinkan perbandingan yang lebih jelas.

3. Visualisasi Data: Diagram Lain

Selain diagram batang, terdapat berbagai jenis visualisasi lain yang dapat digunakan, tergantung pada karakteristik data dan tujuan analisis kita.

a. Diagram Lingkaran (Pie Chart)

Diagram lingkaran sangat cocok untuk menunjukkan proporsi atau persentase dari keseluruhan. Misalnya, kita bisa melihat seberapa besar kontribusi gaji dari setiap posisi terhadap total anggaran gaji.

Untuk membuat diagram lingkaran, kita mengganti fungsi px.bar() dengan px.pie():

```python # Menggunakan DataFrame 'lowker' yang sama atau yang sudah diproses # 'values' menentukan ukuran setiap irisan (misalnya Gaji) # 'names' menentukan label untuk setiap irisan (misalnya Posisi) fig_pie = px.pie(lowker, values='Gaji', names='Posisi', title='Distribusi Gaji berdasarkan Posisi') fig_pie.show() ```

Contoh visualisasi diagram lingkaran yang menampilkan distribusi gaji berdasarkan posisi pekerjaan. Setiap irisan mewakili sebuah posisi, dengan ukuran irisan menunjukkan proporsi gajinya.

Catatan: Meskipun diagram lingkaran dapat digunakan, untuk data lowongan pekerjaan, diagram batang mungkin lebih disukai jika kita ingin membandingkan nilai gaji secara langsung, bukan hanya proporsinya.

b. Diagram Sebar (Scatter Plot)

Diagram sebar digunakan untuk melihat hubungan antara dua variabel numerik. Jika kita memiliki data seperti jumlah pengalaman kerja dan gaji, kita bisa menggunakan scatter plot untuk melihat apakah ada korelasi di antara keduanya.

Latihan Interaktif: Mencoba Diagram Lain

Bagaimana jika kita ingin membuat diagram sebar (scatter plot) dari data lowongan pekerjaan ini? Pikirkan dua variabel dari data yang kita miliki (atau yang bisa kita kumpulkan) yang paling cocok untuk divisualisasikan menggunakan scatter plot. Tuliskan kode Python menggunakan plotly.express.scatter() untuk memvisualisasikannya!

Coba Buat Kode Scatter Plot

Penting untuk memilih jenis visualisasi yang paling sesuai dengan data dan pertanyaan yang ingin Anda jawab. Diagram lingkaran dan diagram sebar, meskipun dapat dibuat, mungkin kurang ideal untuk kasus lowongan pekerjaan dibandingkan dengan diagram batang, tergantung pada fokus analisisnya.
