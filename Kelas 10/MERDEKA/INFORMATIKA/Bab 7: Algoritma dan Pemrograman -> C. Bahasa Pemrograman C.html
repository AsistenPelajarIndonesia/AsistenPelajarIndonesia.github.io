Bab 7: Algoritma dan Pemrograman

C. Bahasa Pemrograman C

Bahasa Pemrograman C, yang sering disingkat sebagai bahasa C, merupakan bahasa pemrograman yang dirancang oleh Dennis M. Ritchie dan Brian W. Kernighan pada awal tahun 1970. Perkembangan bahasa C sangat erat kaitannya dengan sistem operasi UNIX, yang sebagian besar ditulis menggunakan bahasa C.

Karena sejarahnya yang panjang, bahasa C telah memiliki berbagai standar yang mendefinisikan perilakunya. Beberapa standar yang umum dikenal antara lain:

  • Definisi Kernighan & Ritchie (K&R)
  • ANSI-C (X-3.159 -1989-)
  • Definisi AT&T (termasuk superset C, C++)
  • GNU Coding Standards

Di lingkungan PC, terdapat berbagai versi implementasi bahasa C seperti Lattice C, Microsoft C/Microsoft QuickC, dan Turbo C/Borland C++.

Bahasa C sangat populer dan banyak digunakan untuk berbagai keperluan, termasuk pengembangan sistem operasi, program tingkat rendah (yang berinteraksi langsung dengan perangkat keras), pembuatan toolkit pemrograman, dan pengembangan aplikasi. Kelebihan utama bahasa C adalah kemampuannya menghasilkan kode yang singkat, efisien, namun tetap mudah dibaca. Ini membedakannya dari bahasa mesin yang sangat efisien tetapi sulit dibaca, atau bahasa tingkat tinggi lain yang mudah dibaca tetapi kurang efisien. Namun, perlu diakui bahwa membaca program bahasa C membutuhkan latihan lebih dibandingkan bahasa tingkat tinggi lainnya.

1. Membuat Program Pertama dengan Bahasa C

Untuk memulai perjalanan Anda dalam pemrograman C, Anda memerlukan dua hal utama:

  1. Lingkungan Pengembangan: Sebuah tempat untuk menulis, mengkompilasi, dan menjalankan kode program Anda.
  2. Pemahaman Proses Kompilasi: Mengetahui bagaimana kode program yang Anda ketik diubah menjadi program yang dapat dieksekusi oleh komputer.

Secara tradisional, proses pembuatan program melibatkan beberapa tahap:

Sumber Data (Contoh: <span class="keywords">Sample.c</span>) -> Compiler -> Data Objek (Contoh: <span class="keywords">Sample.obj</span>) -> Linker -> File yang Dapat Dieksekusi (Contoh: <span class="keywords">Sample.exe</span>)

Namun, dengan adanya IDE (Integrated Development Environment), proses ini menjadi jauh lebih sederhana. Anda cukup menulis kode, lalu dengan satu klik, program Anda siap dijalankan.

a. Persiapan Lingkungan Pengembangan Program Menggunakan Bahasa C

Untuk membuat program di bahasa C, Anda memerlukan beberapa perangkat lunak:

  • Editor Teks: Tempat Anda mengetikkan kode program (misalnya, dalam file dengan ekstensi .c).
  • Kompilator (Compiler): Menerjemahkan kode bahasa C Anda menjadi bahasa mesin atau bahasa assembly.
  • Assembler: Mengubah bahasa mesin/assembly menjadi kode biner yang dapat dieksekusi komputer.
  • Linker: Menggabungkan berbagai bagian program yang dihasilkan menjadi satu file executable.

Untuk mempermudah, semua perangkat lunak ini seringkali terintegrasi dalam satu paket yang disebut Lingkungan Pengembangan Terpadu (Integrated Development Environment - IDE). Beberapa IDE populer untuk bahasa C antara lain: Eclipse, Atom, Code::Blocks, Geany, dan Visual Studio. Selain IDE yang terpasang di komputer, ada juga IDE berbasis cloud seperti Ideone, tutorialspoint.com, dan onlinegdb.com. Untuk pengembangan di ponsel, ada CppDroid, Mobile C, dan Coding C.

Menguasai lingkungan pengembangan secara maksimal akan meningkatkan produktivitas Anda dalam menghasilkan program yang efisien dan berkualitas tinggi, terutama saat bekerja dalam tim.

Contoh Tampilan IDE (Visual Studio Code):

Membuat Program Pertama: "Halo Dunia!"

Mari kita buat program C pertama kita.

/* Program Pertamaku:
* Membuat program untuk mencetak "Halo Dunia!"
*/
#include<stdio.h>

int main() {
    printf("Halo Dunia!\n");
    return 0;
}

Saat mengetik kode ini, perhatikan setiap karakter. Bahasa C sangat sensitif terhadap kesalahan pengetikan (misalnya, lupa tanda titik koma ; akan menyebabkan error sintaks).

Langkah-langkah Praktis

  1. Buka IDE pilihan Anda (misalnya, Code::Blocks atau Geany).
  2. Pilih File -> New -> Empty File.
  3. Ketikkan kode program "Halo Dunia!" di atas.
  4. Simpan berkas kode Anda dengan nama yang deskriptif, contohnya hello_world.cpp, di direktori yang terorganisir.
  5. Kompilasi program Anda: Pilih Build -> Build atau tekan Ctrl+F9.
  6. Jalankan program Anda: Pilih Build -> Run atau tekan Ctrl+F10.

Jika berhasil, Anda akan melihat keluaran:

Halo Dunia!

Praktik Baik Pemrograman

  • Penamaan File: Selalu berikan nama file yang deskriptif dan gunakan komentar di baris pertama kode untuk menjelaskan isi file tersebut.
  • Komentar Kode: Tulis komentar untuk menjelaskan fungsi dari sebuah blok kode. Komentar tidak dieksekusi oleh kompiler, tetapi sangat membantu dalam pemahaman. Komentar dapat ditulis dalam dua cara:
    • /* Ini adalah komentar */ (bisa multi-baris)
    • // Ini adalah komentar satu baris
  • Struktur Program: Memahami struktur dasar program C, seperti header file (#include <stdio.h>) yang menyediakan fungsi input/output, fungsi utama (int main()) sebagai titik awal eksekusi, dan pernyataan return 0; yang menandakan program berjalan dengan sukses.

2. Struktur Program Bahasa C

Bahasa C adalah bahasa yang terstruktur. Mari kita bedah struktur program "Halo Dunia!" yang sudah kita buat:

#include<stdio.h> // Memasukkan file header untuk fungsi input/output

int main() { // Fungsi utama, titik awal eksekusi program
    printf("Halo Dunia!\n"); // Mencetak teks ke layar, \n untuk baris baru
    return 0; // Menandakan program berakhir dengan sukses
}

Penjelasan detail dari setiap bagian:

  • #include <stdio.h>: Merupakan direktif preprosesor yang memberitahu kompiler untuk menyertakan file header stdio.h. File header ini berisi deklarasi fungsi-fungsi standar untuk input dan output, seperti printf dan scanf.
  • int main() { ... }: Ini adalah fungsi utama program. Eksekusi program selalu dimulai dari sini. Kode di dalam kurung kurawal { } adalah blok kode yang akan dijalankan.
  • printf("Halo Dunia!\n");: Memanggil fungsi printf untuk menampilkan teks "Halo Dunia!" ke layar. Karakter \n (newline) akan memindahkan kursor ke baris berikutnya setelah teks ditampilkan.
  • return 0;: Mengindikasikan bahwa fungsi main telah selesai dieksekusi dengan sukses. Nilai 0 adalah konvensi umum untuk eksekusi yang berhasil.

Modifikasi Program

Anda bisa mencoba memodifikasi program untuk:

  1. Mengubah teks "Halo Dunia!" menjadi "Hello World!".
  2. Mencetak dua baris kalimat, misalnya "Halo Dunia!" dan "Saya siap belajar pemrograman!".

a. Komponen Program dalam Bahasa C

Sebelum membuat program yang lebih kompleks, kita perlu memahami komponen dasar bahasa C:

  1. Kata Kunci (Keyword): Kata-kata yang memiliki makna khusus dan tidak dapat diubah atau digunakan sebagai nama identifier. Bahasa C standar memiliki 32 kata kunci. Contoh: int, float, return, if, else, dll.
  2. Identifier: Nama unik yang Anda berikan pada variabel, fungsi, atau elemen program lainnya. Aturan penulisan identifier:
    • Tidak boleh sama dengan kata kunci.
    • Terdiri dari kombinasi huruf (besar/kecil), angka, dan underscore (_).
    • Harus dimulai dengan huruf atau underscore.
    • Bersifat case-sensitive (misalnya, Nama berbeda dengan nama).

    Praktik Baik Identifier:

    • Hindari penggunaan underscore di awal nama kecuali untuk keperluan khusus.
    • Gunakan huruf kecil untuk nama variabel dan fungsi. Jika terdiri dari beberapa kata, gunakan camel case (misalnya, totalHargaBarang).
    • Hindari penggunaan identifier yang mirip untuk menghindari kebingungan (misalnya, totalHarga vs total_harga).
    • Pilih nama identifier yang mnemonic (mudah diingat dan menggambarkan fungsinya).
    • Konstanta biasanya ditulis dengan huruf kapital (misalnya, PI).
  3. Tipe Data: Menentukan jenis data yang dapat disimpan oleh sebuah variabel dan berapa banyak memori yang dialokasikan. Tipe data dasar dalam C antara lain:
    Nama Tipe Jenis Data Ukuran Memori Rentang Nilai
    intBilangan bulat4 byte-2.1x109 hingga 2.1x109
    shortBilangan bulat2 byte-32768 hingga 32767
    longBilangan bulat8 byte-9.2x108 hingga 9.2x108
    floatBilangan riil (desimal)4 byte1.2x10-38 hingga 3.4x1038
    doubleBilangan riil (desimal)8 byte2.3x10-308 hingga 1.7x10308
    charKarakter1 byte-127 hingga 128 (ASCII)

    Tipe data signed (bawaan) dapat menampung nilai positif dan negatif. Dengan menambahkan kata kunci unsigned di depan tipe data, tipe data hanya akan menampung nilai positif.

    Praktik Baik Tipe Data: Gunakan tipe data yang paling sesuai dengan kebutuhan untuk menghemat memori dan meningkatkan efisiensi.

  4. Variabel: Wadah untuk menyimpan nilai yang dapat berubah selama program berjalan. Variabel harus dideklarasikan dengan tipe data dan nama identifier.
    int totalHarga; // Deklarasi variabel totalHarga bertipe int
    float jarak = 2.28; // Deklarasi dengan nilai awal
    int a, b, c; // Deklarasi beberapa variabel sekaligus

    Variabel dapat bersifat lokal (hanya bisa diakses di dalam fungsi tempat ia dideklarasikan) atau global (dapat diakses di mana saja dalam program).

  5. Konstanta: Nilai yang tidak dapat diubah setelah dideklarasikan. Menggunakan kata kunci const.
    const float PI = 3.14;
  6. Membaca dan Menulis (Input/Output):
    • Fungsi scanf digunakan untuk membaca input dari pengguna.
    • Fungsi printf digunakan untuk menampilkan output ke layar.

    Spesifikasi Format digunakan untuk menentukan bagaimana data dibaca atau ditulis:

    Nama Tipe Spesifikasi Format (Signed) Spesifikasi Format (Unsigned)
    int%d%ud
    short%d%ud
    long%ld%uld
    float%f%ud
    double%lf%uld
    char%c%uc

    Saat menggunakan scanf untuk membaca variabel, gunakan operator alamat & (misalnya, scanf("%d", &bilangan);).

    Untuk menampilkan nilai variabel dengan printf, cukup sertakan spesifikasi formatnya (misalnya, printf("%d\n", bilangan);).

    Karakter khusus seperti \n (newline), \t (tab), dan lainnya disebut escape sequence.

    /* Program Baca Tulis */
    #include<stdio.h>

    int main() {
        int bilangan;
        scanf("%d", &bilangan);
        printf("Bilangan yang dibaca bernilai: %d.\n", bilangan);
        return 0;
    }

    Untuk memformat bilangan riil (float/double), Anda bisa menentukan jumlah digit di belakang desimal, contohnya %.3f untuk menampilkan 3 digit desimal.

    float x = 12.3456789;
    printf("%.3f\n", x); // Output: 12.346
