Bab 7: Algoritma dan Pemrograman

B. Bahasa Pemrograman Prosedural

Belajar bahasa pemrograman itu ibarat belajar bahasa manusia. Kita memulainya dengan mencoba, mengenal, dan menggunakannya untuk keperluan sehari-hari. Kita tidak langsung mempelajari teori bahasa secara mendalam. Seperti saat belajar Bahasa Inggris, kita mulai dengan menyapa, bertanya, atau mengucapkan kalimat sederhana yang sering kita gunakan. Dari contoh-contoh ini, kita belajar kosakata penting dan pola kalimat, misalnya struktur SPOK (Subjek, Predikat, Objek, Keterangan), kalimat tanya, dan sebagainya. Setelah itu, barulah kita beranjak ke konsep yang lebih kompleks dan mempelajari tata bahasanya secara formal, sebelum akhirnya bisa menulis.

Dalam pemrograman, prosesnya juga serupa. Kita belajar dari "membaca" program yang sudah ada, sebelum mencoba "menulis" kode program sendiri. Proses menulis kode (coding) baru dimulai setelah kita memahami pola-pola dari contoh program yang sudah ada. Perbedaan utamanya dengan bahasa manusia adalah, teks program yang kita tulis tidak hanya dibaca dan dipahami, tetapi juga bisa dimengerti oleh mesin dan dieksekusi (dijalankan).

Variabel

Wadah untuk menyimpan data.

Ekspresi

Kombinasi operand dan operator yang menghasilkan nilai.

Struktur Kontrol Keputusan

Mengatur alur program berdasarkan kondisi (jika-maka).

Struktur Kontrol Perulangan

Mengatur program untuk melakukan aksi berulang kali.

Setiap bahasa pemrograman memiliki paradigma, keunggulan, dan tantangannya masing-masing. Di unit ini, kita akan mengenal bahasa pemrograman C, salah satu contoh bahasa pemrograman prosedural. Bahasa C memiliki empat elemen generik yang berlaku di hampir semua bahasa pemrograman prosedural, yaitu:

  • Variabel: Tempat menyimpan nilai.
  • Ekspresi: Kombinasi data dan operator untuk menghasilkan nilai baru.
  • Struktur Kontrol Keputusan: Mengarahkan alur program berdasarkan kondisi.
  • Struktur Kontrol Perulangan: Mengulang blok kode berdasarkan kondisi tertentu.

Memahami keempat elemen dasar ini adalah kunci untuk membuat program, bukan sekadar menguasai satu bahasa pemrograman saja.

C. Bahasa Pemrograman C

Bahasa C dikembangkan oleh Dennis M. Ritchie dan Brian W. Kernighan pada awal tahun 1970-an. Perkembangan C sangat erat kaitannya dengan sistem operasi UNIX, yang sebagian besar ditulis menggunakan C. Seiring waktu, muncul berbagai standar C, seperti K&R, ANSI-C, dan GNU Coding Standards. Di lingkungan PC, kita mengenal varian C seperti Lattice C, Microsoft C, dan Turbo C.

Kelebihan bahasa C menjadikannya sangat populer untuk membuat berbagai jenis program, mulai dari sistem operasi, pemrograman tingkat rendah (yang berinteraksi langsung dengan perangkat keras), toolkit pemrograman, hingga aplikasi umum. C mampu menghasilkan kode yang singkat, efisien, dan relatif mudah dibaca, meskipun tingkat kesulitannya membaca program C bisa jadi lebih tinggi dibandingkan bahasa tingkat tinggi lainnya.

1. Membuat Program Pertama dengan Bahasa C

Sebelum memulai coding, kita perlu mempersiapkan lingkungan pengembangan. Ini termasuk editor teks untuk menulis kode, kompilator (compiler) yang menerjemahkan kode C ke bahasa mesin, assembler yang mengubah bahasa mesin ke kode biner, dan linker yang menyatukan semua komponen menjadi program yang dapat dieksekusi (executable).

(CPU & Layar Pengguna adalah tujuan eksekusi)

Untungnya, kini ada IDE (Integrated Development Environment) yang menyatukan semua alat ini dalam satu paket. Dengan IDE, proses menulis, mengompilasi, dan menjalankan program menjadi jauh lebih mudah, seringkali hanya dengan satu klik!

Contoh tampilan IDE:

// Tampilan jendela IDE dengan berbagai ekstensi pemrograman // (Contoh: TypeScript, C#, Python, C/C++)

IDE modern seperti Visual Studio Code, Eclipse, Atom, atau Code::Blocks sangat membantu. Ada juga IDE berbasis cloud yang bisa diakses dari mana saja.

a. Persiapan Lingkungan Pengembangan Program Menggunakan Bahasa C

Untuk memulai pemrograman C, kita perlu menyiapkan beberapa perangkat lunak:

  • Editor Teks: Tempat kita mengetikkan kode program (misalnya `.c` file).
  • Kompilator (Compiler): Menerjemahkan kode C menjadi bahasa mesin atau assembly.
  • Assembler: Mengubah bahasa mesin menjadi kode biner yang bisa dijalankan komputer.
  • Linker: Menggabungkan berbagai hasil proses sebelumnya menjadi satu file executable.

Saat ini, semua alat tersebut seringkali sudah terintegrasi dalam sebuah Integrated Development Environment (IDE). Beberapa IDE populer untuk C antara lain:

  • Eclipse
  • Atom
  • Code::Blocks
  • Geany
  • Visual Studio

Beberapa IDE juga tersedia secara online (cloud-based) seperti Ideone, tutorialspoint.com, dan onlinegdb.com, atau bahkan untuk perangkat seluler seperti CppDroid, Mobile C, dan Coding C.

Menguasai IDE secara maksimal akan meningkatkan produktivitas dan kualitas program yang kita hasilkan, terutama saat bekerja dalam tim.

Menulis Program Pertama Anda: "Halo, Dunia!"

Mari kita mulai dengan program paling dasar, yaitu "Halo, Dunia!". Program ini akan menampilkan teks tersebut ke layar.

/* Program Pertamaku: * Membuat program untuk mencetak "Halo Dunia!" */ #include <stdio.h> int main() { printf("Halo Dunia!\n"); return 0; }

Langkah-langkahnya:

  1. Buka IDE Anda (misalnya Code::Blocks atau Geany) dan buat file baru (File -> New -> Empty File).
  2. Ketikkan kode di atas dengan cermat. Perhatikan setiap karakter, karena bahasa pemrograman sangat sensitif terhadap kesalahan pengetikan (misalnya lupa titik koma `;` akan menyebabkan error).
  3. Simpan file dengan nama yang deskriptif, misalnya `hello_world.cpp`, dalam direktori yang mudah ditemukan. Struktur folder yang baik sangat penting!
  4. Kompilasi (Build -> Build atau Ctrl+F9) dan jalankan program (Build -> Run atau Ctrl+F10).

Jika semuanya benar, Anda akan melihat output:

Halo Dunia!

Selamat! Anda berhasil menjalankan program pertama Anda.

Praktik Baik Pemrograman: Komentar dan Struktur Kode

Menulis kode yang baik dan mudah dipahami adalah kunci. Berikut beberapa praktik baik:

  • Komentar: Gunakan komentar untuk menjelaskan apa yang dilakukan kode Anda. Komentar tidak dieksekusi oleh compiler. Ada dua cara membuat komentar:
    • /* Komentar ini bisa lebih dari satu baris */
    • // Komentar satu baris
    /* Program Pertamaku: * Membuat program untuk mencetak "Halo Dunia!" */ #include <stdio.h> int main(){ // Fungsi utama program printf("Halo Dunia!\n"); // Mencetak Halo Dunia! ke layar return 0; // Program berjalan dengan benar }
  • Nama File: Sebutkan nama file di baris pertama kode sebagai komentar.
  • Deskripsi Fungsi: Tuliskan tujuan dari potongan kode agar mudah dicari kembali.

Struktur Dasar Program Bahasa C

Mari kita bedah struktur program "Halo, Dunia!":

#include <stdio.h> int main(){ printf("Halo Dunia!\n"); return 0; }
  • #include <stdio.h>: Ini adalah preprocessor directive yang menyertakan file header `stdio.h`. File ini berisi fungsi-fungsi standar input/output, seperti `printf` (untuk menampilkan output) dan `scanf` (untuk menerima input).
  • int main() { ... }: Ini adalah fungsi utama. Setiap program C harus memiliki fungsi `main`. Program akan mulai dieksekusi dari sini. Tanda kurung kurawal `{}` menandai awal dan akhir blok kode dalam fungsi.
  • printf("Halo Dunia!\n");: Memanggil fungsi `printf` untuk menampilkan teks "Halo Dunia!" ke layar. Karakter \n adalah escape sequence yang berarti pindah ke baris baru (newline).
  • return 0;: Mengindikasikan bahwa program telah selesai dieksekusi dengan sukses. Nilai 0 adalah konvensi untuk eksekusi yang berhasil.

Anda bisa memodifikasi program ini, misalnya mengubah teks "Halo Dunia!" menjadi "Hello World!".

Komponen Program dalam Bahasa C

Sebelum mempelajari hal yang lebih kompleks, kita perlu memahami komponen dasar bahasa C:

  1. Kata Kunci (Keyword): Kata-kata yang sudah memiliki makna khusus dalam C dan tidak bisa digunakan sebagai nama identifier. Contohnya `int`, `return`, `if`, `while`, dll. Ada 32 kata kunci standar dalam C.
  2. Identifier: Nama unik yang kita berikan untuk variabel, fungsi, atau elemen program lainnya. Identifier harus mengikuti aturan:
    • Tidak boleh sama dengan kata kunci.
    • Terdiri dari huruf (a-z, A-Z), angka (0-9), dan underscore (_).
    • Harus dimulai dengan huruf atau underscore.
    • Sifatnya case-sensitive (huruf besar dan kecil berbeda maknanya, misal: `nama` berbeda dengan `Nama`).

    Praktik Baik Identifier:

    • Hindari menggunakan nama yang terlalu mirip atau membingungkan (misal: `a` dan `A`, `l` dan `1`).
    • Gunakan nama yang mnemonic (mudah diingat dan menggambarkan tujuannya).
    • Untuk nama variabel atau fungsi yang terdiri dari beberapa kata, gunakan format camelCase (misal: `totalHargaBarang`, `hitungLuasLingkaran`).
    • Hindari nama yang diawali underscore kecuali untuk keperluan khusus.
    • Konstanta biasanya dinamai dengan huruf kapital (misal: `PI`).
  3. Tipe Data: Menentukan jenis data yang dapat disimpan dalam variabel dan berapa banyak memori yang dibutuhkan. Beberapa tipe data dasar dalam C:
    • int: Bilangan bulat (contoh: 10, -5).
    • float: Bilangan desimal (contoh: 3.14, -0.5).
    • double: Bilangan desimal dengan presisi lebih tinggi dari float.
    • char: Satu karakter (contoh: 'a', 'B', '?').

    Setiap tipe data memiliki spesifikasi format yang digunakan saat input/output, contohnya %d untuk int, %f untuk float, dan %c untuk char.

    Contoh penggunaan tipe data:

    #include <stdio.h> int main() { int umur = 17; float tinggi = 1.75; char grade = 'A'; printf("Umur: %d tahun\n", umur); printf("Tinggi: %.2f meter\n", tinggi); // %.2f untuk 2 angka di belakang koma printf("Grade: %c\n", grade); return 0; }

    Outputnya akan menampilkan nilai-nilai tersebut dengan format yang sesuai.

  4. Variabel: Wadah untuk menyimpan nilai yang dapat berubah selama program berjalan. Variabel harus dideklarasikan dengan tipe data dan nama identifier.
    int jumlah; // Deklarasi variabel 'jumlah' bertipe integer float harga = 15000.50; // Deklarasi dan inisialisasi variabel 'harga'

    Variabel yang dideklarasikan di dalam sebuah fungsi disebut variabel lokal dan hanya bisa digunakan di dalam fungsi itu saja. Variabel yang dideklarasikan di luar fungsi disebut variabel global dan bisa diakses di mana saja dalam program.

  5. Konstanta: Mirip variabel, namun nilainya tidak bisa diubah setelah dideklarasikan. Menggunakan kata kunci const.
    const float PI = 3.14159;
  6. Membaca dan Menulis: Menggunakan fungsi scanf() untuk membaca input dari pengguna dan printf() untuk menampilkan output.

    Perhatikan penggunaan %d, %f, %c sebagai spesifikasi format. Saat menggunakan scanf, kita perlu menggunakan tanda & di depan nama variabel untuk memberikan alamat memori.

    #include <stdio.h> int main() { int angka; printf("Masukkan sebuah angka: "); scanf("%d", &angka); // Membaca integer dan menyimpannya di variabel 'angka' printf("Anda memasukkan angka: %d\n", angka); return 0; }
