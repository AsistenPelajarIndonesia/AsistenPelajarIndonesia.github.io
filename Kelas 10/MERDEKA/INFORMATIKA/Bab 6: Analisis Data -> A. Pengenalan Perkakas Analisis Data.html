2. Python: Bahasa Pemrograman untuk Analisis Data

Aktivitas AD-K10-02-P: Mengenal Python

Python adalah bahasa pemrograman yang sangat populer, terutama di bidang analisis data. Bahasa ini menyediakan banyak library yang mempermudah berbagai tahapan analisis.

Bahasa pemrograman pada dasarnya adalah cara kita berkomunikasi dengan komputer untuk memberikan perintah. Ada berbagai macam bahasa pemrograman, namun Python unggul karena kemudahan sintaksisnya dan ekosistem library-nya yang kaya, menjadikannya pilihan utama untuk analisis data.

Dalam bagian ini, kita akan mempelajari dasar-dasar Python yang relevan untuk analisis data. Ingatlah bahwa Python di sini berfungsi sebagai alat bantu. Konsep algoritma dan pemrograman akan sedikit disinggung sebagai latihan, namun fokus utamanya adalah bagaimana Python dapat membantu kita memproses data.

a. Mencetak Data (print)

Perintah paling dasar dalam bahasa pemrograman adalah menampilkan data atau output. Dalam Python, kita menggunakan perintah print().

Anda bisa mengubah parameter di dalam tanda kurung print(). Misalnya, mengganti teks "Hello World" menjadi "Selamat Datang". Setelah kode dieksekusi (dengan menekan tombol Run), program akan menampilkan data sesuai yang Anda masukkan.

Latihan 1: Eksplorasi Perintah Print

  1. Coba ubah kode menjadi print("2 + 3"). Jalankan (Run). Amati hasilnya.
  2. Selanjutnya, coba ubah menjadi print(2 + 3). Apa yang Anda lihat sebagai outputnya?
  3. Sekarang, coba dengan print("2" + "3"). Perhatikan perbedaannya.
  4. Apa makna penggunaan tanda petik ("") di sekitar angka pada contoh nomor 1 dan 3?

Tanda petik ("") digunakan untuk mendefinisikan string, yaitu data berupa teks atau alfanumerik yang tidak dihitung secara aritmatika. Ketika Anda menggunakan "2" + "3", Python akan menggabungkan kedua string tersebut menjadi "23". Namun, jika Anda melakukan operasi aritmatika langsung pada angka (misalnya 2 + 3), hasilnya adalah nilai numerik 5.

Pemrograman Python di sini memiliki kesamaan dengan pemrograman C yang mungkin pernah Anda pelajari di unit Algoritma dan Pemrograman. Pendekatan prosedural juga akan diterapkan pada program-program selanjutnya.

b. Pemberian Nilai Data (Assignment)

Selain mencetak data secara langsung, kita juga bisa menyimpan data ke dalam sebuah wadah bernama variabel, lalu mencetaknya. Proses ini disebut assignment.

Assignment dilakukan dengan cara mendefinisikan sebuah nama variabel, kemudian mengisinya dengan suatu nilai menggunakan tanda sama dengan (=).

Contoh assignment dan operasi penjumlahan sederhana. (Bayangkan di sini ada ilustrasi kode Python yang menunjukkan `bil1 = 10`, `bil2 = 5`, `jumlah = bil1 + bil2`, dan `print(jumlah)` atau hanya `jumlah` di baris terakhir.)

Dalam contoh di atas:

  • Variabel bil1 diisi dengan nilai 10.
  • Variabel bil2 diisi dengan nilai 5.
  • Variabel jumlah diisi dengan hasil penjumlahan bil1 dan bil2.

Untuk menampilkan nilai dari sebuah variabel, Anda cukup menuliskan nama variabel tersebut (terutama jika berada di baris terakhir), atau menggunakan perintah print().

c. Banyak Nilai untuk Satu Variabel (Array)

Selama ini, variabel seperti bil1, bil2, dan jumlah hanya dapat menyimpan satu nilai saja. Jika kita memberikan nilai baru ke variabel yang sama, nilai lama akan tertimpa. Namun, kita juga bisa menyimpan banyak nilai dalam satu variabel menggunakan struktur data yang disebut array (atau list di Python).

Dalam kehidupan sehari-hari, kita sering berhadapan dengan data yang memiliki banyak nilai, misalnya daftar nilai siswa dalam satu mata pelajaran. Di Python, kita bisa membuat array menggunakan tanda kurung siku ([ ]).

Memahami Array di Python

Array memungkinkan kita menyimpan kumpulan nilai. Setiap nilai dalam array memiliki posisi yang disebut indeks, yang dimulai dari 0.

Contoh penggunaan array dan mengakses elemennya. (Bayangkan di sini ada ilustrasi kode Python yang menunjukkan `data = [10, 9, 7, 8, 10, 8]`, `print(data[1])`, `data[2] = 10`, `print(data)`, `data.append(9)`, `print(data)` beserta outputnya.)

Misalnya, pada array data = [10, 9, 7, 8, 10, 8]:

  • Nilai 10 berada di indeks 0.
  • Nilai 9 berada di indeks 1.
  • Nilai 7 berada di indeks 2, dan seterusnya.

Untuk mengakses elemen tertentu dalam array, kita menggunakan kurung siku [ ] setelah nama variabel array, diikuti dengan nomor indeksnya. Contoh: data[1] akan menghasilkan nilai 9. Kita juga bisa mengubah nilai elemen di indeks tertentu (misal data[2] = 10) atau menambahkan elemen baru di akhir array menggunakan metode .append().

d. Penelusuran Data/Pengulangan (Loop: for)

Terkadang, kita perlu memproses setiap elemen dalam sebuah array satu per satu. Di sinilah perintah perulangan atau loop berperan. Salah satu jenis loop yang umum digunakan di Python adalah loop for.

Loop for sangat berguna untuk mengulang suatu blok kode untuk setiap item dalam sebuah urutan (seperti array).

Contoh penggunaan loop `for` untuk menelusuri elemen array. (Bayangkan di sini ada ilustrasi kode Python yang menggunakan loop `for` untuk mencetak setiap elemen array beserta indeksnya.)

Dalam contoh di atas:

  • Kita memiliki array data yang berisi beberapa angka.
  • Kita juga mendefinisikan variabel indeks yang dimulai dari 0.
  • Perintah for elemen in data: akan mengulang blok kode di bawahnya untuk setiap nilai (elemen) yang ada di dalam array data.
  • Di dalam loop, kita mencetak keterangan "Elemen ke", diikuti dengan nilai indeks saat ini dan nilai elemen itu sendiri.
  • Setelah mencetak, nilai indeks dinaikkan sebesar 1 (indeks = indeks + 1) agar pada iterasi berikutnya, indeks yang dicetak sesuai.

Perintah for diawali dengan tanda titik dua (:), dan blok kode yang akan diulang (di dalam loop) ditulis dengan menjorok ke kanan (indentasi) untuk menandai bahwa perintah tersebut merupakan bagian dari loop.

e. Uji Kondisi (Branch: try-except)

Kadang-kadang, sebuah array bisa berisi data yang beragam, tidak hanya bilangan, tetapi juga teks (string). Bagaimana cara kita mengolah data seperti ini?

Misalnya, kita ingin membagi setiap elemen dalam array dengan angka 2. Jika elemen tersebut adalah bilangan, pembagian akan berjalan lancar. Namun, jika elemen tersebut adalah teks, Python akan menghasilkan pesan kesalahan (error).

Contoh pengolahan data bervariasi yang menghasilkan error. (Bayangkan di sini ada ilustrasi kode Python yang mencoba membagi elemen array `[10, 9, "Cindi", 4, "8"]` dengan 2, dan menghasilkan `TypeError`.)

Kesalahan seperti TypeError: unsupported operand type(s) for /: 'str' and 'int' muncul karena kita mencoba melakukan operasi aritmatika (pembagian) antara tipe data yang tidak kompatibel (string dengan integer).

Untuk mengatasi situasi di mana data bisa bervariasi dan mungkin tidak selalu sesuai dengan operasi yang kita inginkan, Python menyediakan mekanisme penanganan kesalahan yang disebut percabangan (branch). Salah satu cara paling umum untuk menangani ini adalah dengan menggunakan blok perintah try dan except.

Cara kerja try-except adalah sebagai berikut:

  • Di dalam blok try, kita mencoba untuk mengeksekusi serangkaian perintah.
  • Jika terjadi kesalahan selama eksekusi di blok try, program akan "melompat" ke blok except dan menjalankan perintah di dalamnya.
  • Jika tidak ada kesalahan yang terjadi di blok try, blok except akan dilewati.

Dengan menggunakan try-except, kita bisa memastikan bahwa program tidak berhenti mendadak ketika menemukan data yang tidak dapat diproses. Sebaliknya, kita bisa memberikan pesan yang informatif kepada pengguna.

Contoh penggunaan `try-except` untuk menangani data bervariasi. (Bayangkan di sini ada ilustrasi kode Python yang menggunakan `try-except` untuk mencoba membagi elemen array dengan 2. Jika berhasil, cetak hasilnya. Jika gagal, cetak "Bukan bilangan".)

Dalam contoh ini, ketika program bertemu dengan "Cindi" atau "8" (yang diperlakukan sebagai string), blok except akan dijalankan, mencetak "Bukan bilangan".

f. Pustaka Kode (Library Import)

Saat kita membutuhkan informasi atau fungsi yang belum pernah kita pelajari atau tidak tersedia dalam materi kita, kita akan mencarinya di perpustakaan. Dalam pemrograman, konsep serupa diwakili oleh pustaka kode atau library.

Library adalah kumpulan kode program atau paket yang menyediakan fungsionalitas tambahan untuk program kita. Library ini dikembangkan oleh penyedia bahasa pemrograman atau komunitas penggunanya. Dengan menggunakan library, kita bisa mendapatkan kemampuan baru tanpa harus menulis semua kode dari awal.

Misalnya, di Python, ada library seperti NumPy untuk operasi numerik yang efisien, Pandas untuk manipulasi dan analisis data tabular, atau Matplotlib untuk visualisasi data. Kita akan mempelajari cara menggunakan library ini di bab-bab selanjutnya.
